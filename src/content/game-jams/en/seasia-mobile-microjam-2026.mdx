## Theme

**Thumb-Only Tension**. Create a game that can be played with one thumb while preserving meaningful decision-making.

## Current Phase

The jam is **currently in progress**. Daily standups, feedback channels, and mentor sessions are active in Discord.

## Constraints

- Portrait mode preferred (landscape allowed with strong justification)
- Target session duration: 60-180 seconds
- At least one progression layer (unlock, score chase, or run modifiers)
- All submissions must run on Android WebView or mobile browser

## Ongoing Schedule

- **Kickoff:** Apr 20, 19:00 SGT
- **Playtest sprint:** Apr 25, 20:00 SGT
- **Submission deadline:** Apr 30, 23:59 SGT

## Prize Track

Total rewards: **SGD 9,000**

- Best Mobile Feel - SGD 3,500
- Best Short Session Loop - SGD 2,500
- Best Monetization-Free Design - SGD 1,500
- Rising Team Award - SGD 1,500

## Submission Process

1. Submit a web-playable link and APK backup.
2. Include a 30-second portrait gameplay clip.
3. Publish your control schema and accessibility settings.
4. Tag entry with `sea-mobile-microjam-2026`.

## Judging Criteria

- Input clarity and responsiveness
- Replayability in short sessions
- Stability on mid-range devices
- Originality within mobile constraints
