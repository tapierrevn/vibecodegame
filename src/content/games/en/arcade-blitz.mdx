Arcade Blitz is a high-tempo arena action game built around movement mastery, fast reads, and combo routing under pressure.

## Gameplay

Each run drops you into compact combat arenas with escalating enemy patterns. You chain dashes, parries, and ability cancels to keep your score multiplier alive while surviving increasingly dense wave compositions.

## Key Features

- Skill-based dodge and counter loop with responsive inputs
- Combo meter that rewards aggressive, clean execution
- Wave modifiers that change enemy behavior every round
- Short-session structure designed for repeat runs and leaderboard climbs

## Why It Stands Out

Arcade Blitz blends classic arcade readability with modern action pacing. Attacks are clearly telegraphed, but decisions still need to happen in split seconds, which keeps each attempt tense and satisfying.

## AI-Assisted Development

`ChatGPT` was used to prototype enemy pattern variants and scoring rules, while `Cursor` accelerated iteration on wave tuning scripts and state-machine refactors.
